## App Test
Application with version RN 0.70.6, it is developed in ts with an atomic design architecture based on reusable components.
	
## Technologies
Project is created with:

* atomic-desing
* redux-toolkit
* style components
* react navigation (V6)
* redux-thunk
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../imaginamos-app
$ npm install
$ cd ios
$ pod install
```

Run app

```
$ npm run ios
$ npm run android

```
